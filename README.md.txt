# 📊 Sales Performance Dashboard – Power BI

## 🔖 Project Title
*Sales Performance Dashboard*

---

## 📌 Overview

This Power BI project delivers an interactive dashboard that provides a clear and concise overview of business sales performance. It showcases key metrics like total profit, sales amount, and top-selling products. The design focuses on simplicity, visual clarity, and user experience, making it ideal for quick analysis and decision-making.

---

## 🧰 Tools & Technologies Used

- *Power BI Desktop*
- *Power Query* for basic data cleaning and shaping
- *Built-in visuals* (Bar Charts, KPI Cards, Slicers)
- No custom DAX used in this version

---

## 📂 Dashboard Layout

### 1️⃣ Page 1: Sales Overview
- KPI cards showing:
  - Total Profit
  - Sales Amount
  - Units Sold
  - Order Count
- Bar chart displaying the *Most Sold Product*
- Slicers for Region and Product filtering

### 2️⃣ Page 2: Product-Wise Performance
- Visualization of product-wise sales or quantity sold
- Helps identify which products are contributing the most
- Simple layout for easy comparison

### 3️⃣ Page 3: Region-Wise Breakdown
- Region-wise sales comparison (e.g., bar or pie chart)
- Highlights top-performing regions
- Filters available for deeper analysis

---

## 🧠 Key Features

- Clean and consistent layout across all pages
- KPI cards for quick metric monitoring
- Product and region-level breakdowns
- Welcome message for better UX
- Light personal branding in footer
- Fully interactive with filters (slicers)
- No DAX — beginner-friendly logic using only Power Query and default visuals

---

## 📊 Sample Insights

- Identify which product had the highest sales
- View overall performance at a glance
- Compare performance across different regions
- Explore data interactively with filters
